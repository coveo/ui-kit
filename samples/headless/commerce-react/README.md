# Headless Commerce React Sample

A React application demonstrating Coveo Headless Commerce controllers.

## Getting Started

```bash
pnpm install
pnpm dev
```

## What's Included

- Commerce search with product list
- Product listing page
- Facets (auto-generated from commerce catalog)
- Sort controls
- Pagination
- Product recommendations

## Credentials

This sample uses the Coveo sample commerce organization. No authentication setup required.
