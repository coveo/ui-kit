# Headless Search React Sample

A React application demonstrating Coveo Headless Search controllers.

## Getting Started

```bash
pnpm install
pnpm dev
```

## What's Included

- Search box with suggestions
- Result list with templates
- Facets (author, source, type)
- Sort controls
- Pager
- Tab navigation

## Credentials

This sample uses the `searchuisamples` public organization. No authentication setup required.
