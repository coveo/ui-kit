# Welcome to the Atomic Storybook

Explore the components below to learn more about how to use Atomic to build modern search experiences.

## Getting Started

- **Atomic**: A web-component library for building modern UIs interfacing with the Coveo platform.
- **Quantic**: A Lightning Web Component (LWC) library for Salesforce search experiences.
- **Headless**: A stateful middle layer between UI elements and the Coveo Platform.
