This component is used to define a section within a layout. 
Each `atomic-layout-section` reserves space based on its section attribute, 
allowing the layout to position and style content consistently within the `atomic-search-layout` or `atomic-commerce-layout`.

An example using the `atomic-commerce-interface` and `atomic-commerce-layout` where we define sections for **search**, **main** content, **status**, **products**, and **pagination**:

```html
<atomic-commerce-interface>

  <atomic-commerce-layout>

    <atomic-layout-section section="search">
        ...
    </atomic-layout-section>
    
    <atomic-layout-section section="main">
        
      <atomic-layout-section section="status">
       ... 
      </atomic-layout-section>
      
      <atomic-layout-section section="products">
       ... 
      </atomic-layout-section>

      <atomic-layout-section section="pagination">
       ... 
      </atomic-layout-section>

    </atomic-layout-section>
  </atomic-commerce-layout>
</atomic-commerce-interface>
```
