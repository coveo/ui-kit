## Usage

The `atomic-numeric-range` component defines a range within an `atomic-numeric-facet`, specifying the start and end values for a numeric filter.

```html
<atomic-numeric-facet field="price">
  <atomic-numeric-range start="0" end="100"></atomic-numeric-range>
  <atomic-numeric-range start="100" end="500"></atomic-numeric-range>
</atomic-numeric-facet>
```
