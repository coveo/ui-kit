## Usage
This component is used to display an icon in the UI. It is used internally for icon rendering and can be used directly in your Atomic layout.

```html
<atomic-icon icon="https://raw.githubusercontent.com/coveo/ui-kit/master/packages/atomic/src/images/arrow-top-rounded.svg"></atomic-icon>
```
