## Usage
The `atomic-aria-live` component notifies screen readers of changes in the search and commerce interfaces. It automatically manages ARIA live regions for accessibility purposes.

This component:
- Creates accessible live regions for screen reader announcements
- Manages polite and assertive announcements based on priority
- Automatically debounces rapid updates to prevent overwhelming screen readers
- Works seamlessly within search and commerce interfaces
