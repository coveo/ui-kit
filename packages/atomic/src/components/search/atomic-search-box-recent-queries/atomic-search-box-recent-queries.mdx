This component is placed inside the [atomic-search-box](/docs/atomic-search-box--docs) to display recent queries as the user types.

By default, atomic-search-box-recent-queries is already included.
You can add it manually to customize its properties.
```html
<atomic-search-box>
  <atomic-search-box-recent-queries></atomic-search-box-recent-queries>
</atomic-search-box>
```

**Note**: This component requires the analytics to be enabled. It will not display any recent queries if analytics are disabled.
