## Usage

The `atomic-result-section-title` component defines the title area within a result template. Use it to display the result's name or page title, making the result list scannable.

```html
<atomic-result-template>
  <template>
    <atomic-result-section-title>
      <atomic-result-link></atomic-result-link>
    </atomic-result-section-title>
  </template>
</atomic-result-template>
```
