
The `atomic-result-badge` component renders a badge to highlight special features of a result.

This component is used within `atomic-result-template` components inside the result list to display badges that emphasize important result attributes.

```html
<atomic-search-interface>
  ...
  <atomic-result-list>
    <atomic-result-template>
      <template>
        <atomic-result-section-badges>
          <atomic-result-badge field="filetype"></atomic-result-badge>
          <atomic-result-badge label="trending"></atomic-result-badge>
          <atomic-result-badge icon="assets://star" label="featured"></atomic-result-badge>
        </atomic-result-section-badges>
      </template>
    </atomic-result-template>
  </atomic-result-list>
  ...
</atomic-search-interface>
```
