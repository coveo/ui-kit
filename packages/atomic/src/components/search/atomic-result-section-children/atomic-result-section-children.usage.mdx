## Usage

The `atomic-result-section-children` component defines the children area within a result template. Use it to display folded (nested) results, typically rendered via the `atomic-result-children` component.

```html
<atomic-result-template>
  <template>
    <atomic-result-section-children>
      <atomic-result-children>
        <atomic-result-children-template>
          <template>
            <atomic-result-link></atomic-result-link>
          </template>
        </atomic-result-children-template>
      </atomic-result-children>
    </atomic-result-section-children>
  </template>
</atomic-result-template>
```
