The `atomic-result-html` component renders the HTML value of a string result field.

**⚠️ Security Warning:** There is an inherent XSS security concern associated with the usage of this component. Use only with fields for which you are certain the data is harmless.

This component is used within `atomic-result-template` components inside the result list:

```html
<atomic-search-interface>
  ...
  <atomic-result-list>
    <atomic-result-template>
      <template>
        <atomic-result-html field="excerpt"></atomic-result-html>
        <atomic-result-html field="custom_html_field" sanitize="true"></atomic-result-html>
      </template>
    </atomic-result-template>
  </atomic-result-list>
  ...
</atomic-search-interface>
```
