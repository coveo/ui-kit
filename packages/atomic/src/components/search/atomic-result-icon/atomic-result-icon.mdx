The `atomic-result-icon` component outputs an icon corresponding to the file type or object type of a search result. It automatically selects the appropriate icon based on result metadata, or displays a generic document icon if no match is found.

```html
<atomic-result-template>
  <template>
    <atomic-result-section-visual>
      <atomic-result-icon></atomic-result-icon>
    </atomic-result-section-visual>
  </template>
</atomic-result-template>
```
