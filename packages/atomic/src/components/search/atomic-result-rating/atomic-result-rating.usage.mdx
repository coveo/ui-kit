
The `atomic-result-rating` component displays the rating of a result as a set of icons (stars by default).

This component is used within `atomic-result-template` components inside the result list:

```html
<atomic-search-interface>
  ...
  <atomic-result-list>
    <atomic-result-template>
      <template>
        <atomic-result-rating field="snrating"></atomic-result-rating>
      </template>
    </atomic-result-template>
  </atomic-result-list>
  ...
</atomic-search-interface>
```
