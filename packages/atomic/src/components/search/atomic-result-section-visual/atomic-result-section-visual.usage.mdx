## Usage

The `atomic-result-section-visual` component defines the visual area within a result template. Use it to display images, icons, or avatars that provide a visual representation of the result. It maintains a 1:1 aspect ratio with a fixed size that varies by image size, layout, density, and screen size.

```html
<atomic-result-template>
  <template>
    <atomic-result-section-visual>
      <atomic-result-image field="thumbnailurl"></atomic-result-image>
    </atomic-result-section-visual>
  </template>
</atomic-result-template>
```
