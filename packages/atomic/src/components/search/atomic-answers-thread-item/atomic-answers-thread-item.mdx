The `atomic-answers-thread-item` component renders a single generated answers thread item with a timeline dot and optional vertical line. It can be collapsible and displays its content in the default slot.

> **Internal:** This component is intended for internal use and to be used along the `atomic-generated-answers-thread` component.

## Basic Usage

```html
<atomic-answers-thread-item
  title="what are they for"
  is-collapsible
  is-expanded
>
  <div>Safeguards against misinformation and bias in AI-generated snippets.</div>
</atomic-answers-thread-item>
```

## Properties

- `title`: The title displayed for the thread item.
- `is-collapsible`: Whether the thread item can be expanded or collapsed.
- `is-expanded`: Whether the thread item is initially expanded.
- `hide-line`: Whether the timeline line should be hidden (e.g., for the last item).

## Slots

- `(default)`: The content rendered when the item is expanded.

## Styling

Use the exposed parts to style the component:

- `item`
- `timeline`
- `timeline-dot`
- `timeline-line`
- `header`
- `title`
- `title-button`
- `content`
