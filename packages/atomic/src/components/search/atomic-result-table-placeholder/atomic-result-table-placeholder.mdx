## Usage (Internal Component)

The `atomic-result-table-placeholder` component is an internal component that provides an intermediate visual state rendered before the first results are available in a table layout. It is automatically rendered by `atomic-result-list` when configured with `display="table"` during the loading state.

This component is not intended for direct use in application code.

### Rendered automatically in result lists

```html
<atomic-search-interface>
  <atomic-search-layout>
    <atomic-layout-section section="main">
      <atomic-result-list display="table">
        <!-- atomic-result-table-placeholder is rendered automatically during loading -->
        <atomic-result-template>
          <template>
            <atomic-result-section-title>
              <atomic-result-link></atomic-result-link>
            </atomic-result-section-title>
          </template>
        </atomic-result-template>
      </atomic-result-list>
    </atomic-layout-section>
  </atomic-search-layout>
</atomic-search-interface>
```

### Properties

The component accepts the following properties (automatically set by `atomic-result-list`):

- `density`: Display density (`'comfortable'`, `'normal'`, `'compact'`)
- `imageSize`: Image size (`'large'`, `'small'`, `'icon'`, `'none'`)
- `rows`: Number of placeholder rows to render
