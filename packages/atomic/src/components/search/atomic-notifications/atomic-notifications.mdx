The `atomic-notifications` component displays notifications generated by the Coveo Search API through [Triggers](https://docs.coveo.com/en/1458). Notifications are messages that can be configured to appear when specific query conditions are met.

This component is typically placed within the "status" section of the search interface layout.

```html
<atomic-search-interface>
  ...
  <atomic-search-layout>
    ...
    <atomic-layout-section section="main">
      ...
      <atomic-layout-section section="status">
        
        <atomic-notifications></atomic-notifications>

      </atomic-layout-section>
    </atomic-layout-section>
  </atomic-search-layout>
</atomic-search-interface>
```
