## Usage

The `atomic-result-section-title-metadata` component defines the title metadata area within a result template. Use it to display fields directly related to the title of the result, such as ratings or source indicators, rendered in a small font size just beneath the title.

```html
<atomic-result-template>
  <template>
    <atomic-result-section-title-metadata>
      <atomic-result-text field="author"></atomic-result-text>
      <atomic-result-text field="source"></atomic-result-text>
    </atomic-result-section-title-metadata>
  </template>
</atomic-result-template>
```
