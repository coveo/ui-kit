This component is typically placed within the "facets" section of the layout.

You can also place facets inside an `atomic-facet-manager` component to ensure the effective functioning of DNE (Dynamic Navigation Experience). 
More info here: [atomic-facet-manager](https://docs.coveo.com/en/2918).

```html
<atomic-search-interface>
  ...
  <atomic-search-layout>
    ...
    <atomic-layout-section section="facets">
      ...
        <atomic-facet field="author" label="Authors"></atomic-facet>

    </atomic-layout-section>
  </atomic-search-layout>
</atomic-search-interface>
```
