## Usage

The `atomic-result-section-excerpt` component defines the excerpt area within a result template. Use it to render an informative summary of the result's content, displayed with a fixed height of one to three lines depending on layout and density.

```html
<atomic-result-template>
  <template>
    <atomic-result-section-excerpt>
      <atomic-result-text field="excerpt"></atomic-result-text>
    </atomic-result-section-excerpt>
  </template>
</atomic-result-template>
```
