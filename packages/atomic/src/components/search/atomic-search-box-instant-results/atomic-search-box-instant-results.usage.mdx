
This component is placed inside the [atomic-search-box](/docs/atomic-search-box--docs) to display instant results as the user types.

The `atomic-search-box-instant-results` is not included in the `atomic-search-box` by default.

This component needs the `atomic-search-box-query-suggestions` component to be present in the same `atomic-search-box` component.
```html
<atomic-search-box>
  <atomic-search-box-query-suggestions></atomic-search-box-query-suggestions>
  <atomic-search-box-instant-results></atomic-search-box-instant-results>
</atomic-search-box>
```
