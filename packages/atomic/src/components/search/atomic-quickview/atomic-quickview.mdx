The `atomic-quickview` component renders a button which the end user can click to open a modal box containing a preview of a result.

This component must be used inside an `atomic-result-template` component.

```html
<atomic-search-interface>
  <atomic-result-list>
    <atomic-result-template>
      <template>
        <atomic-quickview></atomic-quickview>
      </template>
    </atomic-result-template>
  </atomic-result-list>
</atomic-search-interface>
```

### Common Use Cases
* Provide users a quick preview of document content without leaving the search results page.
* Allow users to scan through multiple documents efficiently to determine relevance before clicking through.
* Display PDF or HTML content in a modal for immediate review.
