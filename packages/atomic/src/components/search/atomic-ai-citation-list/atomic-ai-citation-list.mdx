The `atomic-ai-citation-list` component provides buttons that allow the end user to navigate through the different result pages.

This component should be placed within an `atomic-search-interface` component, usually in the "pagination" section of the layout.

**Note:** Use either `atomic-ai-citation-list` OR `atomic-load-more-results` for pagination, but not both. These components provide alternative pagination approaches:
- `atomic-ai-citation-list`: Provides traditional page-based navigation with numbered pages. It is often used along with the `atomic-results-per-page` component.
- `atomic-load-more-results`: Allows users to incrementally load additional results with a "Load More" button.

```html
<atomic-search-interface>
  ...
  <atomic-search-layout>
    ...
    <atomic-layout-section section="main">
      ...
      <atomic-layout-section section="pagination">

        <atomic-ai-citation-list></atomic-ai-citation-list>

      </atomic-layout-section>
    </atomic-layout-section>
  </atomic-search-layout>
</atomic-search-interface>
```

## Customization

You can customize the number of page buttons displayed and the icons used for the previous/next buttons:

```html
<atomic-ai-citation-list >
</atomic-ai-citation-list>
```

The component automatically hides when there are no search results or when there's an error state.
