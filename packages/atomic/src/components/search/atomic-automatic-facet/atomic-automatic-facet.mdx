The `atomic-automatic-facet` component is a special type of facet generated by the automatic facets feature.
Unlike regular facets, which need to be explicitly defined and requested in the query, automatic facets are dynamically generated by the index in response to the query.

## Important Note

**This component should never be used on its own.** It is used internally by the `atomic-automatic-facet-generator` component to automatically render updated facets. However, you can use the shadow parts to style the automatically generated facets.

To learn more about the automatic facet generator feature, see [About the Facet Generator](https://docs.coveo.com/en/n9sd0159/).

## Styling

The `atomic-automatic-facet` exposes shadow parts that allow you to customize the appearance of automatically generated facets:

- `facet` - The wrapper for the entire facet
- `label-button` - The button that displays the label and allows expanding/collapsing the facet
- `label-button-icon` - The label button icon
- `clear-button` - The button that resets the actively selected facet values
- `clear-button-icon` - The clear button icon
- `values` - The facet values container
- `value-label` - The facet value label
- `value-count` - The facet value count
- `value-checkbox` - The facet value checkbox
- `value-checkbox-checked` - The checked facet value checkbox
- `value-checkbox-label` - The facet value checkbox clickable label
- `value-checkbox-icon` - The facet value checkbox icon

### Example: Styling Automatic Facets

```css
atomic-automatic-facet::part(facet) {
  border-radius: 8px;
  background-color: #f5f5f5;
}

atomic-automatic-facet::part(label-button) {
  color: #333;
  font-weight: 600;
}

atomic-automatic-facet::part(value-checkbox) {
  accent-color: #0066cc;
}
```

## Usage

While you don't directly add `atomic-automatic-facet` to your search interface, you control its behavior through the `atomic-automatic-facet-generator` component.

See the `atomic-automatic-facet-generator` documentation for more information on how to configure automatic facets.
