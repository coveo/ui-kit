## Usage

The `atomic-result-section-badges` component defines the badges area within a result template. Use it to render labels that highlight special features of a result, such as "NEW" or "FEATURED".

```html
<atomic-result-template>
  <template>
    <atomic-result-section-badges>
      <span class="badge">NEW</span>
    </atomic-result-section-badges>
  </template>
</atomic-result-template>
```
