The `atomic-numeric-facet` component displays a facet of the results for the current query as numeric ranges.

This component is typically placed within the "facets" section of the layout.

You can also place facets inside an `atomic-facet-manager` component to ensure the effective functioning of DNE (Dynamic Navigation Experience).

```html
<atomic-search-interface>
  ...
  <atomic-search-layout>
    ...
    <atomic-layout-section section="facets">
      ...
      <atomic-numeric-facet field="ytviewcount" label="View Count"></atomic-numeric-facet>
    </atomic-layout-section>
  </atomic-search-layout>
</atomic-search-interface>
```

## With Custom Input

You can enable a custom numeric input that allows users to set their own ranges:

```html
<atomic-numeric-facet 
  field="price" 
  label="Price"
  with-input="integer"
></atomic-numeric-facet>
```

Use `with-input="decimal"` for fields with decimal values:

```html
<atomic-numeric-facet 
  field="rating" 
  label="Rating"
  with-input="decimal"
></atomic-numeric-facet>
```

## With Manual Ranges

You can define manual ranges using `atomic-numeric-range` child elements:

```html
<atomic-numeric-facet field="price" label="Price">
  <atomic-numeric-range start="0" end="100" label="Under $100"></atomic-numeric-range>
  <atomic-numeric-range start="100" end="500" label="$100 to $500"></atomic-numeric-range>
  <atomic-numeric-range start="500" end="1000" label="$500 to $1000"></atomic-numeric-range>
  <atomic-numeric-range start="1000" end="1000000" label="Over $1000"></atomic-numeric-range>
</atomic-numeric-facet>
```

## Display as Links

By default, facet values are displayed as checkboxes (allowing multiple selection). You can display them as links instead (single selection):

```html
<atomic-numeric-facet 
  field="ytviewcount" 
  label="View Count"
  display-values-as="link"
></atomic-numeric-facet>
```
