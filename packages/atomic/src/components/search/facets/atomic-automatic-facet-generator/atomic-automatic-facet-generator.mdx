The `atomic-automatic-facet-generator` component renders facets from the automatic facets feature. Unlike regular facets, which need to be explicitly defined and requested in the query, automatic facets are dynamically generated by the index in response to the search query.

This component renders one or more [atomic-automatic-facet](https://docs.coveo.com/en/atomic/latest/reference/components/atomic-automatic-facet/) based on the `desiredCount` property.

To learn more about the automatic facet generator feature, see: [About the Facet Generator](https://docs.coveo.com/en/n9sd0159/).

This component is typically placed within the facets section of the search interface layout.
```html
<atomic-search-interface>
  ...
  <atomic-search-layout>
    ...
    <atomic-layout-section section="facets">

      <atomic-automatic-facet-generator desired-count="5" number-of-values="8"></atomic-automatic-facet-generator>

    </atomic-layout-section>
  </atomic-search-layout>
</atomic-search-interface>
```
