
The atomic-external component allows components defined outside of an Atomic search interface to initialize with it.

```html

<atomic-external>
  <atomic-search-box></atomic-search-box>
</atomic-external>
 
<!-- ... -->
 
<atomic-search-interface>
  <atomic-result-list>
    <!-- ... -->
  </atomic-result-list>
</atomic-search-interface>
```
