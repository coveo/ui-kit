This component displays facet values as segmented buttons in a horizontal layout, ideal for a small number of mutually exclusive options.

This component is typically placed within the "facets" section of the layout or within an `atomic-facet-manager` component.

```html
<atomic-search-interface>
  ...
  <atomic-search-layout>
    ...
    <atomic-layout-section section="facets">
      ...
        <atomic-segmented-facet field="filetype" label="File Type"></atomic-segmented-facet>

    </atomic-layout-section>
  </atomic-search-layout>
</atomic-search-interface>
```
