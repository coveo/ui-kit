## Usage

The `atomic-ipx-embedded` component is used to create embedded IPX (In-Product Experience) interfaces. It provides a container with three slot areas for header, body, and footer content.

```html
<atomic-ipx-embedded>
  <div slot="header">
    <h2>Your Header</h2>
  </div>
  <div slot="body">
    <p>Your main content goes here</p>
  </div>
  <div slot="footer">
    <button>Action</button>
  </div>
</atomic-ipx-embedded>
```

## Slots

- **header**: Content to display in the header area
- **body**: Main content area
- **footer**: Footer content area (optional)

## Features

- Automatically detects if footer slot has content
- Generates a unique ID if one is not provided
- Supports responsive breakpoints
- Integrates with `atomic-ipx-body` for consistent styling
