The `atomic-ipx-button` component displays a floating button that toggles the visibility of the IPX modal. This button is typically positioned at the bottom-right corner of the screen and provides users with quick access to the IPX search experience.

## Usage

This component must be used alongside `atomic-ipx-modal` within an `atomic-search-interface`:

```html
<atomic-search-interface>
  <atomic-ipx-modal>
    <!-- Modal content -->
  </atomic-ipx-modal>
  <atomic-ipx-button label="Help"></atomic-ipx-button>
</atomic-search-interface>
```
