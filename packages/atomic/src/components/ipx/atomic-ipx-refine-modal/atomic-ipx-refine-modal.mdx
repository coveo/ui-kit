This `atomic-ipx-refine-modal` is an internal component. It is publicly documented for the sake of exposing its CSS properties and shadow parts so that you can customize its styles.

You should never insert this component directly in your markup or modify its attributes.

The correct way to use the refine modal in IPX is through the `atomic-ipx-refine-toggle` component.

```html
<atomic-search-interface>
  <atomic-ipx-modal is-open="">
    <atomic-ipx-refine-toggle></atomic-ipx-refine-toggle>
    <!-- Facets defined here will be cloned into the modal when opened -->
    <atomic-layout-section section="facets">
      <atomic-facet field="author" label="Author"></atomic-facet>
      <atomic-facet field="source" label="Source"></atomic-facet>
      <atomic-facet field="filetype" label="File Type"></atomic-facet>
    </atomic-layout-section>
  </atomic-ipx-modal>
</atomic-search-interface>
```

## Related Components

- `atomic-ipx-refine-toggle`: The toggle button that opens the IPX refine modal
- `atomic-facet`: Individual facet components that can be placed in the refine modal
