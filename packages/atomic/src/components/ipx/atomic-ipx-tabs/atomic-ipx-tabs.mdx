# Atomic IPX Tabs

The `atomic-ipx-tabs` component is a wrapper around `atomic-tab-bar` for In-Product Experience (IPX) interfaces. It provides a simple container for organizing `atomic-ipx-tab` elements.

## Internal Component

This component is marked as `@internal` and is primarily used within Coveo's In-Product Experience implementations.

## Usage

Place `atomic-ipx-tab` elements inside the `atomic-ipx-tabs` component to create a tabbed interface:

```html
<atomic-search-interface>
  <atomic-ipx-tabs>
    <atomic-ipx-tab label="All" name="all" expression=""></atomic-ipx-tab>
    <atomic-ipx-tab
      label="Documentation"
      name="documentation"
      expression="@source=documentation"
    ></atomic-ipx-tab>
    <atomic-ipx-tab
      label="Articles"
      name="articles"
      expression="@source=articles"
    ></atomic-ipx-tab>
  </atomic-ipx-tabs>
</atomic-search-interface>
```
