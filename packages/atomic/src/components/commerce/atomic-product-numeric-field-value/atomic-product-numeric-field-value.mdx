## Usage
The `atomic-product-numeric-field-value` component displays a numeric field value for a product, such as ratings or reviews.

It is typically used within `atomic-product-template` components to show numeric attributes of a product in a product list.

```html
<atomic-commerce-interface>
  ...
  <atomic-commerce-product-list>
    <atomic-product-template>
      <template>
        ...
          <atomic-product-numeric-field-value field="ec_rating"></atomic-product-numeric-field-value>

      </template>
    </atomic-product-template>
  </atomic-commerce-product-list>
</atomic-commerce-interface>
```
