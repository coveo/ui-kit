This component should not be used directly. It is internally used by the `atomic-commerce-refine-toggle` component to display the refine modal.
To use the refine modal, you should use the `atomic-commerce-refine-toggle` component instead. It usually is placed within the "status" section of the layout.
```html
<atomic-commerce-interface>
  ...
  <atomic-commerce-layout>
    ...
    <atomic-layout-section section="main">
      ...
      <atomic-layout-section section="status">

            <atomic-commerce-refine-toggle></atomic-commerce-refine-toggle>

      </atomic-layout-section>
    </atomic-layout-section>
  </atomic-commerce-layout>
</atomic-commerce-interface>
```
