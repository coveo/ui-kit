## Usage

The `atomic-product-section-metadata` component defines the metadata area within a product template. Use it to display product attributes such as brand, category, or ratings.

```html
<atomic-product-template>
  <template>
    <atomic-product-section-metadata>
      <atomic-product-text field="ec_brand"></atomic-product-text>
      <atomic-product-rating field="ec_rating"></atomic-product-rating>
    </atomic-product-section-metadata>
  </template>
</atomic-product-template>
```
