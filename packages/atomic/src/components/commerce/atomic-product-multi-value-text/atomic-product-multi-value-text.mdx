## Usage
The `atomic-product-multi-value-text` component displays multiple values for a specific field of a product, such as sizes or colors. It is typically used within `atomic-product-template` components to show product attributes that can have multiple values.

It is typically used within `atomic-product-template` components to display product attributes that can have multiple values, such as sizes or colors.

```html
<atomic-commerce-interface>
  ...
  <atomic-commerce-product-list>
    <atomic-product-template>
      <template>
        ...
          <atomic-product-multi-value-text field="cat_available_sizes"></atomic-product-multi-value-text>

      </template>
    </atomic-product-template>
  </atomic-commerce-product-list>
</atomic-commerce-interface>
```
