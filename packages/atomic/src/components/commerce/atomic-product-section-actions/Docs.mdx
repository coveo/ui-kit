# Product Sections

<Subtitle >
The product section components are the building blocks for creating product templates in commerce interfaces. Each section handles a specific aspect of product presentation, from visual elements to metadata and actions.
</Subtitle>


## Usage 
These components are typically used within an `atomic-product-template` to create cohesive product displays.
They work together to present a complete product view, including images, descriptions, metadata, and actions.

```html
<atomic-product-template>
  <template>
    <atomic-product-section-badges>
      <span class="badge">NEW</span>
    </atomic-product-section-badges>
    
    <atomic-product-section-visual>
      <atomic-product-image field="ec_images"></atomic-product-image>
    </atomic-product-section-visual>
    
    <atomic-product-section-name>
      <atomic-product-text field="ec_name"></atomic-product-text>
    </atomic-product-section-name>
    
    <atomic-product-section-metadata>
      <atomic-product-text field="ec_brand"></atomic-product-text>
      <atomic-product-text field="ec_price"></atomic-product-text>
    </atomic-product-section-metadata>
    
    <atomic-product-section-description>
      <atomic-product-text field="ec_description"></atomic-product-text>
    </atomic-product-section-description>
    
    <atomic-product-section-actions>
      <button class="add-to-cart">Add to Cart</button>
    </atomic-product-section-actions>
    
    <atomic-product-section-bottom-metadata>
      <span>Free shipping • 2-day delivery</span>
    </atomic-product-section-bottom-metadata>
  </template>
</atomic-product-template>
```
