## Usage

The `atomic-product-section-actions` component defines the actions area within a product template. Use it to display interactive elements such as "Add to Cart" buttons or other call-to-action controls.

```html
<atomic-product-template>
  <template>
    <atomic-product-section-actions>
      <button class="add-to-cart">Add to Cart</button>
    </atomic-product-section-actions>
  </template>
</atomic-product-template>
```
