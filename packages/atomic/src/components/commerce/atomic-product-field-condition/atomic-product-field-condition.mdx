## Usage

`atomic-product-field-condition` conditionally renders its content based on product attributes.
It can check whether fields are defined, undefined, match or not match specific values.

It is typically used within `atomic-product-template`.

```html
<atomic-product-template>
  ... 

  <atomic-product-field-condition
    attributes-if-defined="ec_name"
 
  >
  <span>Render me if <strong>ec_name</strong> is defined.</span>
  </atomic-product-field-condition>
  
</atomic-product-template>
```
