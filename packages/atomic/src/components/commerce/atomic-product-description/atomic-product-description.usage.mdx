
The `atomic-product-description` component is used to display a detailed description of a product.
It uses the `ec_shortdesc` property by default but you can change it to `ec_description` with the field attribute.

It is typically used within `atomic-product-template` components to display product information in a concise format.

```html
<atomic-commerce-interface>
  ...
  <atomic-commerce-product-list>
    <atomic-product-template>
      <template>
        ...
        <atomic-product-section-description>
          <atomic-product-description field="ec_description"></atomic-product-description>
        </atomic-product-section-description>

      </template>
    </atomic-product-template>
  </atomic-commerce-product-list>
</atomic-commerce-interface>
```
