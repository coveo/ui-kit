## Usage

The `atomic-product-section-badges` component defines the badges area within a product template. Use it to display labels such as "NEW", "SALE", or "BESTSELLER" that highlight product attributes.

```html
<atomic-product-template>
  <template>
    <atomic-product-section-badges>
      <span class="badge">NEW</span>
    </atomic-product-section-badges>
  </template>
</atomic-product-template>
```
