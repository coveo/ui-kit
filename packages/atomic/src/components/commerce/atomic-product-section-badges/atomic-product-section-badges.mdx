## Usage
This section is meant to render badges that highlight special features of the product.
In commerce interfaces, badges are used to draw attention to product qualities like "NEW", "SALE", "BESTSELLER", or discount percentages.

### Example within a product template
```html
<atomic-commerce-product-list>
  <atomic-product-template>
    <template>
      <!-- ... -->
      <atomic-product-section-badges>
        <span class="badge badge-primary">NEW</span>
        <span class="badge badge-warning">SALE</span>
      </atomic-product-section-badges>
      <!-- ... -->
    </template>
  </atomic-product-template>
</atomic-commerce-product-list>
```

### CSS Variables
The component exposes the following CSS variable for styling consistency:
- `--line-height`: The current line height that child elements should align to

### Styling Guidelines
- Child badges should use `height: var(--line-height)` for consistent sizing
- The section uses flexbox with gap spacing, so individual margins between badges are not needed
- Badges are displayed horizontally but will wrap if space is limited
- Keep badge text concise and impactful for maximum visual effect
