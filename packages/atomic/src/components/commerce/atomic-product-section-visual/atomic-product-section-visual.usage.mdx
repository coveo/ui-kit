## Usage

The `atomic-product-section-visual` component defines the visual area within a product template. Use it to display product images or other visual media.

```html
<atomic-product-template>
  <template>
    <atomic-product-section-visual>
      <atomic-product-image field="ec_images"></atomic-product-image>
    </atomic-product-section-visual>
  </template>
</atomic-product-template>
```
