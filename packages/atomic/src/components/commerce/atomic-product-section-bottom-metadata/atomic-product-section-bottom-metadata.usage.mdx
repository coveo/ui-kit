## Usage

The `atomic-product-section-bottom-metadata` component defines the bottom metadata area within a product template. Use it to display supplementary information such as shipping details or delivery estimates.

```html
<atomic-product-template>
  <template>
    <atomic-product-section-bottom-metadata>
      <span>Free shipping</span>
    </atomic-product-section-bottom-metadata>
  </template>
</atomic-product-template>
```
