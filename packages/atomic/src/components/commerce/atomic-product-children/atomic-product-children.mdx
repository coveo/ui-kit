The `atomic-product-children` component renders a list of child products associated with a parent product. 
It is typically used within `atomic-product-template` components to display related products in a product list.
It is also typically wrapped in `atomic-product-section-children` to provide a section for child products.

This component leverages the [product grouping](https://docs.coveo.com/en/l78i2152/coveo-for-commerce/product-grouping) feature. 

```html
<atomic-commerce-interface>
  ...
  <atomic-commerce-product-list>
    <atomic-product-template>
      <template>
        ...
        <atomic-product-section-children>
          <atomic-product-children></atomic-product-children>
        </atomic-product-section-children>

      </template>
    </atomic-product-template>
  </atomic-commerce-product-list>
</atomic-commerce-interface>
```
