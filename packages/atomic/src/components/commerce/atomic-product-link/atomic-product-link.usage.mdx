
The `atomic-product-link` component renders a product **ec_name**  a clickable link. This component is used within `atomic-product-template` components inside the product list:

```html
<atomic-commerce-interface>
  ...
  <atomic-commerce-product-list>
    <atomic-product-template>
      <atomic-product-link></atomic-product-link>
    </atomic-product-template>
  </atomic-commerce-product-list>
</atomic-commerce-interface>
```
