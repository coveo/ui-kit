## Usage

The `atomic-product-section-description` component defines the description area within a product template. Use it to display product descriptions or detailed text content.

```html
<atomic-product-template>
  <template>
    <atomic-product-section-description>
      <atomic-product-text field="ec_description"></atomic-product-text>
    </atomic-product-section-description>
  </template>
</atomic-product-template>
```
