## Usage

This component is not meant to be used directly in the layout. You should use the `<atomic-commerce-facets>` component instead, which will automatically render the appropriate facets.

```html
<atomic-commerce-interface>
  ...
  <atomic-layout-section section="facets">
    <atomic-commerce-facets></atomic-commerce-facets>
  </atomic-layout-section>
</atomic-commerce-interface>
```
</AtomicDocTemplate>
