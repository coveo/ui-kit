## Usage

The `atomic-insight-full-search-button` component displays a button that links to the full search interface. It is typically positioned in the corner of the insight panel to allow users to navigate to a more complete search experience.

```html
<atomic-insight-interface>
  <atomic-insight-full-search-button tooltip="Open full search"></atomic-insight-full-search-button>
</atomic-insight-interface>
```
