The `atomic-insight-no-results` component is designed to display search tips when no results are found for a search query. It provides helpful suggestions to guide users in refining their search criteria.

This component is typically placed within the main content area of an Insight interface layout:

```html
<atomic-insight-interface>
  ...
  <atomic-insight-no-results></atomic-insight-no-results>
  ...
</atomic-insight-interface>
```

The component automatically displays when a search returns no results and shows:
- A magnifying glass icon
- A message indicating no results were found (with the query highlighted if present)
- Search tips to help users refine their query

Any additional content slotted inside the component will be displayed alongside the default message and tips:

```html
<atomic-insight-no-results>
  <p>Contact support if you need additional assistance.</p>
</atomic-insight-no-results>
```
