The `atomic-insight-pager` component provides buttons that allow the end user to navigate through the different result pages in an insight panel.

This component should be placed within an `atomic-insight-interface` component.

```html
<atomic-insight-interface>
  ...
  <atomic-insight-pager></atomic-insight-pager>
</atomic-insight-interface>
```

## Customization

You can customize the number of page buttons displayed:

```html
<atomic-insight-pager number-of-pages="7"></atomic-insight-pager>
```

The component automatically hides when there is only one page of results or when there's an error state.
