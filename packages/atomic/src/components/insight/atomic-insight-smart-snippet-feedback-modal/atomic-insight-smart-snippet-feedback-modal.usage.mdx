## Usage

The `atomic-insight-smart-snippet-feedback-modal` component displays a modal dialog that allows users to provide feedback on a smart snippet answer. It is automatically created by the `atomic-insight-smart-snippet` component when the user clicks the feedback button, so you typically do not need to add it manually.

```html
<atomic-insight-interface>
  <atomic-insight-smart-snippet></atomic-insight-smart-snippet>
</atomic-insight-interface>
```
