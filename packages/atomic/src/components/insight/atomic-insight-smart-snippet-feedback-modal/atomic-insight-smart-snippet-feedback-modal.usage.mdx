## Usage

The `atomic-insight-smart-snippet-feedback-modal` component displays a modal dialog that allows users to provide feedback on a smart snippet answer. It is automatically created as a child of the `atomic-insight-interface` when the `atomic-insight-smart-snippet` component is initialized.

```html
<atomic-insight-interface>
  <atomic-insight-smart-snippet></atomic-insight-smart-snippet>
  <atomic-insight-smart-snippet-feedback-modal></atomic-insight-smart-snippet-feedback-modal>
</atomic-insight-interface>
```
