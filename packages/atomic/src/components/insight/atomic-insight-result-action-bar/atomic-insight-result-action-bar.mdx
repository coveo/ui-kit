This component is used to display action buttons within an Insight result template. Action buttons placed in this component appear when the user hovers over a result.

The action bar is positioned absolutely within the result and becomes visible on hover. When the action bar is empty (no slotted content), it automatically hides itself.

Place the `atomic-insight-result-action-bar` inside an `atomic-insight-result-template`:

```html
<atomic-insight-result-template>
  <template>
    <atomic-result-section-title>
      <atomic-result-link></atomic-result-link>
    </atomic-result-section-title>
    <atomic-result-section-excerpt>
      <atomic-result-text field="excerpt"></atomic-result-text>
    </atomic-result-section-excerpt>
    <atomic-insight-result-action-bar>
      <button>Copy Link</button>
      <button>Share</button>
    </atomic-insight-result-action-bar>
  </template>
</atomic-insight-result-template>
```
