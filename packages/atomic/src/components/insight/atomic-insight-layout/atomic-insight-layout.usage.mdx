This component organizes the insight page layout. 
Use `atomic-layout-section` to define the different sections of the page, such as **search**, **facets**, and **results**.

```html
<atomic-insight-layout>
  <atomic-layout-section section="search"></atomic-layout-section>
  <atomic-layout-section section="results"></atomic-layout-section>
</atomic-insight-layout>
```
