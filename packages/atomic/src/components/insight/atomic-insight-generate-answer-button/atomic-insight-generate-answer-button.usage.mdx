## Usage

The `atomic-insight-generate-answer-button` component renders a button that triggers answer generation in the insight interface. It appears when there are results but no active query, allowing agents to generate an AI-powered answer on demand.

```html
<atomic-insight-interface>
  <atomic-insight-generate-answer-button tooltip="Generate Answer"></atomic-insight-generate-answer-button>
</atomic-insight-interface>
```
