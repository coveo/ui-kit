## Usage

The `atomic-insight-user-actions-modal` component displays a modal containing the timeline of actions performed by a specific user. It is automatically created by the `atomic-insight-user-actions-toggle` component when it initializes, so you typically do not need to add it manually.

```html
<atomic-insight-interface>
  <atomic-insight-user-actions-toggle
    user-id="exampleUserId"
    ticket-creation-date-time="2024-01-01T00:00:00Z"
  ></atomic-insight-user-actions-toggle>
</atomic-insight-interface>
```
