The `atomic-insight-history-toggle` component displays a button that toggles the visibility of a history panel. This is useful for showing users their interaction history within an insight interface.

This component should be placed within the `atomic-insight-interface`

```html
<atomic-insight-interface>
  <atomic-insight-history-toggle
    tooltip="View history"
  ></atomic-insight-history-toggle>
</atomic-insight-interface>
```

## Click Callback

You can provide a callback function to handle the button click:

```js
const historyToggle = document.querySelector('atomic-insight-history-toggle');
historyToggle.clickCallback = () => {
  // Handle history panel toggle
  console.log('History toggle clicked');
};
```
