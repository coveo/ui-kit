
The `atomic-insight-tab` component represents an individual tab within the insight interface. Tabs allow users to filter results by applying a constant query expression. Each tab must be placed inside an `atomic-insight-tabs` container.

```html
<atomic-insight-interface>
  <atomic-insight-tabs>
    <atomic-insight-tab label="All" expression="" active></atomic-insight-tab>
    <atomic-insight-tab label="Articles" expression='@documenttype==("Article")'></atomic-insight-tab>
  </atomic-insight-tabs>
</atomic-insight-interface>
```
