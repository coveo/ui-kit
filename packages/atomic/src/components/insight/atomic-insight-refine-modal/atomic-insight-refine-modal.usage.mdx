This component should not be used directly. It is internally used by the `atomic-insight-refine-toggle` component to display the refine modal.
To use the refine modal, you should use the `atomic-insight-refine-toggle` component instead. It usually is placed within the "status" section of the layout.
```html
<atomic-insight-interface>
  ...
  <atomic-insight-layout>
    ...
    <atomic-layout-section section="main">
      ...
      <atomic-layout-section section="status">

            <atomic-insight-refine-toggle></atomic-insight-refine-toggle>

      </atomic-layout-section>
    </atomic-layout-section>
  </atomic-insight-layout>
</atomic-insight-interface>
```

## Related Components

- `atomic-insight-refine-toggle`: The toggle button that opens the refine modal
- `atomic-insight-facet`: Individual facet components that can be placed in the refine modal
- `atomic-breadbox`: Shows active filters and can be used alongside the refine modal
