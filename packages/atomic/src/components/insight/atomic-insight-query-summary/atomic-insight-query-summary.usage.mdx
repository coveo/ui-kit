This component is an internal component used within the Insight Panel interface. It displays information about the current range of results or shows a message about insights related to the current case when no query is active.

When a query is active, it displays the typical query summary (e.g., "Results 1-10 of 100 for search term"). When no query is active and there is no error, it shows "Insights related to this case".

```html
<atomic-insight-interface>
  ...
  <atomic-insight-query-summary></atomic-insight-query-summary>
  ...
</atomic-insight-interface>
```
