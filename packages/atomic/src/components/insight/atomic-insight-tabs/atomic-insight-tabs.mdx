The `atomic-insight-tabs` component wraps a list of `atomic-insight-tab` elements and manages their layout and display.
This is an internal component used within the Insight Panel interface to organize content into different tabbed views.

## Usage

The `atomic-insight-tabs` component should contain one or more `atomic-insight-tab` elements.
Each tab element defines a label and an optional Coveo query expression to filter results when that tab is active.

```html
<atomic-insight-interface>
  <atomic-insight-tabs>
    <atomic-insight-tab
      label="All"
      expression=""
      active="true"
    ></atomic-insight-tab>
    <atomic-insight-tab
      label="Videos"
      expression="@ytchanneltitle"
    ></atomic-insight-tab>
    <atomic-insight-tab
      label="Documentation"
      expression='@documenttype==("WebPage")'
    ></atomic-insight-tab>
  </atomic-insight-tabs>
</atomic-insight-interface>
```

## Related Components

- `atomic-insight-tab`: Individual tab elements that are slotted into this component
- `atomic-insight-interface`: The parent interface component
