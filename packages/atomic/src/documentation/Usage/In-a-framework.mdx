<Meta title="Usage/In a framework/Introduction" />

To use the Atomic library in [React](https://reactjs.org/) or [Angular](https://angular.io/) projects, Coveo offers the [Atomic React wrapper](https://static.cloud.coveo.com/atomic/v3/storybook/index.html?path=/docs/usage-in-a-framework-use-the-react-wrapper--docs) and the [Atomic Angular wrapper](https://static.cloud.coveo.com/atomic/v3/storybook/index.html?path=/docs/usage-in-a-framework-use-the-angular-wrapper--docs), respectively.

Out of the box, you can also use [Atomic in a Vue project](https://static.cloud.coveo.com/atomic/v3/storybook/index.html?path=/docs/usage-in-a-framework-usage-in-a-vue-project--docs).
