<Meta title="Usage/In a framework/Introduction" />

To use the Atomic library in [React](https://reactjs.org/) or [Angular](https://angular.io/) projects, Coveo offers the [Atomic React wrapper](https://docs.coveo.com/en/atomic/latest/usage/frameworks/atomic-react-wrapper/) and the [Atomic Angular wrapper](https://docs.coveo.com/en/atomic/latest/usage/frameworks/atomic-angular-wrapper/), respectively.

Out of the box, you can also use [Atomic in a Vue project](https://docs.coveo.com/en/atomic/latest/usage/frameworks/atomic-vue/).
