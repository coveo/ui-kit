An _Atomic project_ is a directory which contains all the files to build a [Coveo Atomic](https://docs.coveo.com/en/lcdf0264/) search page.
The articles in this section explain how to create, manage, and use an [Atomic](https://docs.coveo.com/en/lcdf0264/) project.

You can [create an Atomic project](https://docs.coveo.com/en/atomic/latest/usage/manage-project/create-an-atomic-project/) locally using the Coveo CLI, or you can create a hosted search page in the [Coveo Administration Console](https://docs.coveo.com/en/183/).
If you create your search page using the [Administration Console](https://docs.coveo.com/en/183/), you can [use the Coveo CLI to download it](https://docs.coveo.com/en/atomic/latest/usage/manage-project/cli-download/) so that you can work with it locally.

Regardless of whether you created your [Atomic](https://docs.coveo.com/en/lcdf0264/) project locally or downloaded it from the [Administration Console](https://docs.coveo.com/en/183/), you can use the Coveo CLI to [deploy it to the [Coveo Platform](https://docs.coveo.com/en/atomic/latest/usage/manage-project/cli-deploy/)(https://docs.coveo.com/en/186/) infrastructure].
Finally, you can leverage a web component to [use a hosted search page](https://docs.coveo.com/en/atomic/latest/usage/manage-project/use-hosted-page/) in your own infrastructure.
